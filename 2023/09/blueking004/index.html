初始化邮件通知

需要注意的是，文中说的测试接口：

用法一：
{
“bk_app_code”:”“,
“bk_app_secret”:”“,
“bk_username”: “admin”,
“receiver”: “****@qq.com”,
“sender”: “houming@domain.cn“,
“title”: “This is a Test”,
“content”: “Welcome to Blueking“
}

用法二：
{
“bk_app_code”:”“,
“bk_app_secret”:”“,
“bk_username”: “admin”,
“receiver”: “****@qq.com”,
“sender”: “蓝鲸houming@domain.cn“,
“title”: “This is a Test”,
“content”: “Welcome to Blueking“
}
